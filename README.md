
## Required Software
- [Python 3.10](https://www.python.org/downloads/) or newer
- [Node.js 14.16 LTS](https://nodejs.org/) or newer (For Tailwind.CSS)
- [Git](https://git-scm.com/)


## how to run - backend

```bash
1 Clone This Repo
2 cd backend
3 pip install -r requirements.txt 
4 python manage.py runserver
```

## how to run - frontend

```bash
1 Clone This Repo
2 cd frontend
3 npm install
4 npm start
```
